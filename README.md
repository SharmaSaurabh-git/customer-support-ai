# customer-support-ai
Customer support ai Project 

## 📌 Overview
This project is a simple AI-based customer support system that generates responses to customer complaints.

## 🚀 Features
- Handles damaged product complaints
- Handles delivery delay issues
- Responds to angry or upset customers
- Provides polite and professional replies

## 🛠️ Tech Used
- Python
- Gradio

## ▶️ How to Run
1. Install dependencies:
   pip install gradio

2. Run the app:
   python app.py
